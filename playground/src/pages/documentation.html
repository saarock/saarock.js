Add the following in your <head> and script:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/saarock/saarock.js@main/src/styles/toast.css">
<script type="module">
  import { saarock } from "https://cdn.jsdelivr.net/gh/saarock/saarock.js@main/dist/index.js";
</script>

üî• How to Use Toast

Example of success toast:

saarock.success({
  position: "right",
  message: "Success! Your form was submitted.",
  comeFrom: "LEFT",
  timeOut: 5000,
  leaveFrom: "RIGHT",
  goTo: "/dashboard"
});

Other types:

saarock.warning({ message: "This is a warning!" });
saarock.error({ message: "An error occurred!" });
saarock.loading({ message: "Loading, please wait..." });

üîù How to Use BackToTop

Add this in your script:

saarock.backToTop({
  backColor: "green" // customize the button color
});

üß© Integration with WordPress

Use wp_enqueue_script and wp_enqueue_style to load the library:

function load_saarock_js() {
  wp_enqueue_style('saarock-toast-style', 'https://cdn.jsdelivr.net/gh/saarock/saarock.js@main/src/styles/toast.css');
  wp_enqueue_script('saarock-js', 'https://cdn.jsdelivr.net/gh/saarock/saarock.js@main/dist/index.js', [], null, true);
}
add_action('wp_enqueue_scripts', 'load_saarock_js');
